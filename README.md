# Задание
## Сборка из строк из прочтений
Реализовать сборку из набора строк строки “генома”. Ввод - ряд последовательностей прочтений в формате fasta. Вывод - собранная строка формата fasta.

Input: in.fa

Output: out.fa

Sample_input: 

>1

TCGT

>2

GTAA

Sample_output:

>out

TCGTAA
